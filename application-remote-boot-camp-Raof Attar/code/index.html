What is a cookie?

Most people would say that a cookie is a form of a sweet bakery. However if you ask a Web developer you will get a different answer. The coming chapter will not discuss the common understanding of a cookie, it will explain what a cookie means in the world of web development.

Technical meaning of a cookie

HTTP cookies are described as a small blocks data created when a user browses a web. This data is placed on the users computer or other device used buy the web browser.
